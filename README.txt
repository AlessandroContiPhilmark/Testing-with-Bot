Si bro, Il corso di Fingeco è un pò una rottura, quindi per usare questa roba:

Installa Node.js e npm sul pc

Apri prompt su questa cartella e scrivi: npm install

Dentro index.js vai a scrivere:
	il path per chrome.exe
	il tuo username e password di Fingeco

Assicurati che command propt abbia l'opzione "Modalità modifica rapida" disattivata

Apri prompt su questa cartella e scrivi: npm start

Vai a guardare anime mentre il programma fa il corso per te.
